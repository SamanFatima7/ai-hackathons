## 🚀 The Projects

### 1. Urdu Legal E-Commerce Agent — ⚖️🇵🇰
*RAISE YOUR HACK · Team: Binary Ninjas*

A multi-agent system that makes everyday Pakistani legal help accessible to millions of Urdu speakers. The platform combines a **Retriever Agent** (fetches relevant clauses), **Parse & Simplify Agent** (explains them in simple Urdu), **Risk Advisor Agent** (flags hidden risks), and a **Draft Notice Agent** (auto-generates legal notices) — all over local Urdu legal texts like the Rent Act and Contract Act.

Built for small landlords, tenants, and small businesses who can't afford a lawyer but still need to understand what they're signing. All conversations are saved in a reusable user profile (stored as a knowledge graph), enabling personalized follow-ups.

**Stack:** Llama · Groq API · LangChain · vector embeddings · Streamlit

### 2. Python Buddy — AI Master 🐍
*Co-Creating with GPT-5 · Team: Code Commanders*

An interactive AI tutor that teaches users to *think like a programmer*, not just memorize syntax. Built around a two-step learning loop:

1. **Learn tab** — structured curriculum with bite-sized Python lessons. If you get a question wrong, an **Adaptive AI Teacher** re-explains the concept in a completely new way.
2. **Practice Arena** — a library of coding challenges. Stuck? A **Socratic Hint System** analyzes your code and asks guiding questions instead of giving you the answer. Once you solve it correctly, the **AI Code Coach** reviews your working code, refactors it to a professional Pythonic standard, and explains every improvement line by line.

Designed to take a beginner from zero to writing code like a seasoned professional.

**Stack:** GPT-5 · Python · Streamlit · structured prompt engineering

🔗 **[Project page on lablab.ai →](https://lablab.ai/ai-hackathons/co-creating-with-gpt-5/code-commanders/ai-master)**

### 3. Predictive Network Maintenance — for Off-Grid Schools 🛰
*AIstronauts: Space Agents on a Mission · Team: AI Astronauts*

An AI system that monitors network performance in off-grid schools and predicts outages or hardware failures *before* they happen. Includes:
- **Predictive maintenance** — ML models that flag failing hardware ahead of time
- **Transparent procurement optimization** — recommends cost-effective, sustainable equipment choices
- **Energy-efficient network management** — load balancing tied to renewable supply
- **Intelligent bandwidth management** — for shared classroom connections

Built to give administrators of off-grid schools the same operational visibility that urban IT teams take for granted.

## 🧭 Why hackathons?

Three reasons. First — the **deadline pressure** strips away analysis paralysis. You ship or you don't. Second — the **team composition** is randomized in a way it never is at work, which makes you a faster communicator. Third — the **judging panel** forces you to articulate the *why* of your project in 90 seconds, which is exam-grade training for everything else you'll do in tech.

Also they're fun. That matters.
